Prompt Engineering thuc chien: viet prompt ro rang, dung muc tieu

Topic: Prompt Engineering Author: Huynh Thanh Xuan 8 min read

Huong dan viet prompt thuc chien de tang do on dinh va kha nang tai su dung trong san pham.

1. Why this topic matters

Prompt la giao dien truc tiep giua con nguoi va LLM.

Prompt tot giup giam hallucination, tiet kiem token va de bao tri.

2. Core idea

Mot prompt hieu qua thuong co 4 thanh phan: vai tro, muc tieu, rang buoc, dinh dang output.

Cang cu the, model cang de tra loi dung huong.

3. Step-by-step breakdown

  1. Mo ta ro vai tro va task cua model.
  2. Dat quy tac output (schema, do dai, ngon ngu).
  3. Them vi du mau va test voi nhieu tinh huong.

4. Example

Prompt tom tat ticket support:

Ban la tro ly CS. Tom tat van de trong 3 gach dau dong, neu thieu thong tin thi danh dau Can bo sung.

5. Common mistakes

  • Prompt dai nhung mo ho.
  • Khong khoa dinh output format.
  • Khong co bo test hoi quy sau moi lan sua.

6. Key takeaways

  • Prompt engineering la cong viec thiet ke he thong, khong chi la viet cau lenh.
  • Nen quan ly prompt nhu code: version, test, review.
  • Ket hop prompt + eval moi cho ket qua on dinh.
